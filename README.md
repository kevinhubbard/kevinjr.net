# Portfolio


My portfolio/blog was made using Node, Express, Handlebars and MySQL.  
Follows the MVC architecture.


### Some of the required packages include:
* Bootstrap
* JQuery
* Body Parser
* Express
* Mongoose
* Multer
* Socket.io
